== Description ==

Add detailed post review block with unlimited rated criterias, negatives and positives lists, summary, affiliate buy button and more to any of your WordPress blog posts and display automatic total rating on single post page and blog listing pages.

This is a Free light version of Ultimate Post Review plugin and include this features:

*   Fastest review plugin
*   Automatic and manual integration
*   Percentage rating system
*   Post review block display (Single post)
*   Review title and summary display
*   Positives and negatives display (up to 5 in free version)
*   Review criterias with separate rating (up to 3 in free version)
*   Automatic summary review rating display
*   Unlimited styles with Custom Styling for all elements
*   Shortcodes and custom functions for custom placing review blocks anywhere in your theme
*   Crossbrowser compatibility (work smooth in any modern browser)
*   Reviews are displayed to visitors in a friendly format
*   WP Multisite and Multiuser (WPMU / WPMS / WordPress MU) compatible
*   Detailed Documentation guide included
*   Minimalist, lightweight, and efficient fast code
*   Work on mobile/touch devices
*   Retina support
*   100% Responsive all plugin elements and pages
*   Translation ready (POT file included)
